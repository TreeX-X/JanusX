# SwitchX 图标系统

黑橙配色的"X"主题图标集合，按用途分类。

## 📁 目录结构

```
icons/
├── logos/              # 品牌标志
├── app-icons/          # 应用图标
└── badges/             # 徽章状态指示器
```

## 🎨 logos/ - 品牌标志

### logo-circle.html
- 圆形主标志
- 黑色背景 + 橙色粗线条 X
- 适用于：圆形头像、社交媒体、Favicon

### logo-square.html
- 方形主标志（圆角 32px）
- 适用于：应用启动页、文档封面

### logo-horizontal.html
- 横版标志（X图标 + SwitchX 文字）
- 适用于：网站 Header、文档页眉、宣传物料

### logo-x-only.html
- 极简纯 X（无背景）
- 橙色渐变效果
- 适用于：水印、Loading 动画、小尺寸图标

## 📱 app-icons/ - 应用图标

### app-icon-macos.html
- macOS 风格应用图标（圆角 44px）
- 渐变背景 + 光晕效果 + 高光
- 尺寸规格：1024×1024（原始）

### app-icon-windows.html
- Windows 11 风格（圆角 18px）
- 锐利线条 + 扁平设计
- 尺寸规格：256×256

## 🏷️ badges/ - 徽章状态指示器

### badge-active.html
- 三种小徽章：圆形激活、圆形未激活、方形激活
- 适用于：Tab 指示器、状态标记

### badge-status.html
- 四种状态：连接中（动画）、已连接、错误、断开
- 颜色映射：
  - 橙色 `#ff7830` - 正常/激活
  - 红色 `#ff5555` - 错误
  - 灰色 `#555` - 断开/未激活

## 🎯 使用规范

### 配色标准
- 主橙色：`#ff7830`
- 渐变起点：`#ff8844`
- 背景黑：`#0a0a0a` / `#1a1a1a`
- 边框透明度：10-30%

### 尺寸建议
| 用途 | 推荐尺寸 |
|---|---|
| Favicon | 32×32, 64×64 |
| macOS 应用图标 | 1024×1024 |
| Windows 应用图标 | 256×256 |
| 文档水印 | 120×120 |
| 状态徽章 | 80×80 |

### 导出 PNG
所有 HTML 图标可通过浏览器开发者工具或截图工具导出为 PNG：

1. 双击打开 HTML 文件
2. F12 开发者工具 → 右键 `<svg>` → Copy → Copy element
3. 粘贴到矢量编辑器（Figma/Sketch）导出 PNG
4. 或使用浏览器截图扩展直接截取
